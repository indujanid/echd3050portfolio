Analysis of Breast Cancer Wisconsin Data Set

Me and my team use python and scikitlearn to analyze Kaggle's Breast Cancer Wisconsin data set to see if we can classify tumors as malignant or benign from features extracted from breast cancer images and identify the features of a tumor (texture, radius, etc) that best predict cancer. We evaluate several machine learning classification models to see which features contribute the most to breast cancer.
